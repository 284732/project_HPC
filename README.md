# project_HPC

**High Performance Computing Project**

**Authors:** Andrea Predieri, Giacomo Michelini, Cattabriga

This repository contains the project developed for the **High Performance Computing** course as part of the **Master's Degree in Digital Automation Engineering** at the **University of Modena and Reggio Emilia (UNIMORE)**.

The project is organized into three consecutive phases, following the progression of the course. The material starts from the fundamental differences between **C++** and **Fortran 90** for scientific computing and parallel programming, and gradually introduces increasingly advanced HPC concepts, culminating in complete parallelization case studies representative of real scientific applications.

Each phase is self-contained and includes its own documentation, theoretical background, and implementation examples.
